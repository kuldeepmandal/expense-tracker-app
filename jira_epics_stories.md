# Spendly - Jira Epics and User Stories

## Epic 1: Authentication & User Management
**Description:** As a user, I want to be able to securely register, login, and manage my profile.
* **Story 1.1:** As a new user, I want to sign up using my full name, email, and password.
* **Story 1.2:** As an existing user, I want to log in to access my Dashboard.
* **Story 1.3:** As a user, I want to securely log out of the application.
* **Story 1.4:** As a user, I want to be able to change my password from the Profile page.
* **Story 1.5:** As a user, I want to set a currency preference (NPR, USD) in my account settings.

## Epic 2: Dashboard & Financial Overview
**Description:** As a user, I want a summary of my financial status immediately upon logging in.
* **Story 2.1:** As a user, I want to see my Total Budget, Total Spent, and Remaining Available balance.
* **Story 2.2:** As a user, I want to see a chart showing the breakdown of my expenses by category.
* **Story 2.3:** As a user, I want to see a list of my 4-5 most recent transactions natively on the dashboard.

## Epic 3: Transaction Logging
**Description:** As a user, I want to easily add, edit, and categorize my expenses and incomes.
* **Story 3.1:** As a user, I want to add a new transaction specifying amount, type (expense/income), category, and date.
* **Story 3.2:** As a user, I want an option to provide extra details like Description, Payment Method, and Notes.
* **Story 3.3:** As a user, I want to view a complete, paginated list of all my historical transactions.
* **Story 3.4:** As a user, I want to search transactions by description or recipient.
* **Story 3.5:** As a user, I want to filter my transactions by date range, category, and type.

## Epic 4: Budgeting & Reports
**Description:** As a user, I want to set spending limits to visualize where my money is going.
* **Story 4.1:** As a user, I want to set an overall monthly budget limit.
* **Story 4.2:** As a user, I want to allocate specific limits to categories (e.g., limit Food to Rs. 5000).
* **Story 4.3:** As a user, I want visual progress bars showing how much of a category limit has been spent.
* **Story 4.4:** As a user, I want to receive alerts when I am close to or over my set budget limits.
